# EvoFit Landing Page Updater

## Description
Skill for updating the EvoFit Trainer landing page (app/page.tsx) with marketing copy.

## When to Use
- When updating marketing messaging on the landing page
- When adding new features to the feature showcase
- When updating pricing or promotional copy
- When refreshing the landing page for campaigns

## Context
- **File:** app/page.tsx (Next.js 14 client component)
- **Framework:** React 18, Tailwind CSS 3.4, Framer Motion
- **Icons:** lucide-react
- **Brand:** Blue (#2563eb), "Transform Your Fitness Business"
- **Pricing Model:** Pay once, own forever (lifetime license)
- **Target Audience:** Independent personal trainers (25-45 years)

## Brand Voice
- Professional but approachable
- Active, direct, benefit-focused
- Use: "Transform", "Elevate", "Professional", "Powerful"
- Avoid: Technical jargon, hyperbole without proof

## Current Sections
1. Hero (headline, sub-headline, CTA)
2. Features (8 feature cards with icons)
3. Feature Spotlight (detailed feature blocks)
4. Benefits (checklist with checkmarks)
5. Pricing (lifetime license card)
6. Social Proof (platform stats)
7. Roles (Admin, Trainer, Client cards)
8. CTA (final call to action)
9. Footer

## Key Stats (always keep updated)
- 1,344 exercises with GIF demonstrations
- 10 body parts, 29 equipment types, 26 target muscles
- 8 program types, 7 set types, 3 difficulty levels
- ACWR training load monitoring
- 5 appointment types, 5 client statuses
