# EvoFit Help File Generator

## Description
Generate user help files, FAQ pages, and documentation from docs/businesslogic.md.

## When to Use
- When creating user-facing help documentation
- When building FAQ pages
- When creating onboarding guides
- When generating feature documentation

## Source Document
docs/businesslogic.md (Version 2.0, March 2026)

## Output Formats

### Help Article Template
```
# [Feature Name]

## Overview
[1-2 paragraph description]

## How It Works
[Step-by-step guide with numbered steps]

## Key Features
- [Feature 1]
- [Feature 2]

## Frequently Asked Questions
**Q: [Common question]**
A: [Clear answer]

## Tips & Best Practices
- [Tip 1]
- [Tip 2]

## Related Features
- [Link to related feature]
```

### FAQ Page Template
```
# Frequently Asked Questions

## Getting Started
### How do I create an account?
[Answer from businesslogic.md onboarding section]

## [Feature Category]
### [Question]
[Answer]
```

## Instructions
1. Read docs/businesslogic.md for authoritative feature information
2. Extract relevant section for the requested help topic
3. Rewrite in user-friendly language (no technical jargon)
4. Add step-by-step "How To" instructions
5. Include screenshots references where available
6. Format consistently using the templates above
